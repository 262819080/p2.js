p2.Body(options)

A physics body.

Parameters

Object options

Description

Options:
- mass: A number >= 0. If zero, the body becomes static. Default is 0.
- position (vec2)
- velocity (vec2)
- angle (vec2)
- angularVelocity (vec2)
- force (vec2)
- angularForce (vec2)
